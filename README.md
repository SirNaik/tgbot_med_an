# üî¨ Telegram Bot for Medical Analysis Interpretation

This Telegram bot uses GigaChat AI to interpret medical laboratory test results. Users can upload documents in various formats (DOC, DOCX, XLS, PDF, JPEG) and receive AI-powered explanations of their test results.

## üìã Features

- Accepts multiple file formats (DOC, DOCX, XLS, PDF, JPEG)
- Uses GigaChat AI for medical analysis interpretation
- Provides information on abnormal values and their possible causes
- Gives recommendations for additional tests if needed
- Includes disclaimer about consulting healthcare professionals

## üìã Requirements

- Python 3.8+
- Telegram Bot Token
- GigaChat API credentials
- python-dotenv (included in requirements.txt)

## üîß Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd <repository-directory>
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Set up environment variables:
```bash
cp .env.example .env
```
Then edit `.env` with your actual credentials.

## ‚öôÔ∏è Configuration

You need to obtain the following:

1. **Telegram Bot Token**: Talk to [@BotFather](https://t.me/BotFather) on Telegram to create a bot and get your token.

2. **GigaChat Credentials**: Obtain API credentials from GigaChat.

Add these to your `.env` file:
```
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
GIGACHAT_CREDENTIALS=your_gigachat_credentials_here
GIGACHAT_SCOPE=your_gigachat_scope_here
```

## ‚ñ∂Ô∏è Running the Bot

After setting up the environment variables, run the bot:

```bash
python bot.py
```

## üìñ Usage

1. Start a chat with your bot on Telegram
2. Send `/start` command to get the welcome message
3. Upload a document with your medical analysis results
4. Wait for the bot to process and interpret your results

## ‚ö†Ô∏è Important Disclaimer

The interpretations provided by this bot are generated by artificial intelligence and are for informational purposes only. They should not be considered as medical advice. Always consult with qualified healthcare professionals for proper diagnosis and treatment.

## üìÅ Supported File Formats

- Documents: DOC, DOCX
- Spreadsheets: XLS, XLSX
- PDF files: PDF
- Images: JPG, JPEG, PNG (Note: For images, OCR functionality would need to be implemented for text extraction)

## üîÑ How It Works

1. User uploads a document with medical test results
2. The bot extracts text from the document based on its format
3. The extracted text is sent to GigaChat with a specific prompt
4. GigaChat analyzes the results as a medical professional would
5. The bot receives and sends back the interpretation to the user
6. A disclaimer is included reminding users to consult healthcare professionals
